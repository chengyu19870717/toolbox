## 一、项目背景

### 1.1 业务场景

银行信贷审批系统使用 mxGraph 格式的 XML 文件定义审批流程。当前存在以下问题：

1. **流程配置不透明**：XML文件存储在数据库字段中，业务人员无法直观查看流程配置
2. **问题排查困难**：流程执行异常时，无法快速定位节点配置问题
3. **文档维护成本高**：流程变更后需手动更新文档，容易遗漏

### 1.2 目标用户

| 角色 | 使用场景 |
|------|---------|
| 业务人员 | 查看流程审批路径、节点配置、授权规则 |
| 运维人员 | 问题排查、配置核对 |
| 开发人员 | 流程变更确认、配置迁移验证 |

### 1.3 核心价值

- **可视化呈现**：将XML转为可读的表格和流程图
- **一键导出**：支持Excel导出，便于归档和分享
- **配置翻译**：将编码转换为可读文本，降低理解门槛

## 二、数据结构分析

### 2.1 XML文件结构

#### 2.1.1 单流程文件（如 195_195.xml）

```xml
<?xml version="1.0" encoding="UTF-8"?>
<mxGraphModel>
  <root>
    <Workflow ... />
    <mxCell id="0"/>
    <mxCell id="1" parent="0"/>
    <mxCell id="2" value="开始" ...>
      <Node ... />
    </mxCell>
    <mxCell id="6" source="2" target="3" ...>
      <Line ... />
    </mxCell>
    ...
  </root>
</mxGraphModel>
```

#### 2.1.2 多流程文件（如 flow_content.xml）

```xml
<mxGraphModel>...</mxGraphModel>
<mxGraphModel>...</mxGraphModel>
<mxGraphModel>...</mxGraphModel>
```

**特点**：无根节点包裹，多个流程定义直接拼接。

### 2.2 核心数据实体

```
┌─────────────────────────────────────────────────────────────┐
│                        mxGraphModel                          │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐          │
│  │  Workflow   │  │    Node     │  │    Line     │          │
│  │  (1个)      │  │   (N个)     │  │   (M条)     │          │
│  └─────────────┘  └─────────────┘  └─────────────┘          │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 输入预处理

```python
def preprocess_xml_content(xml_content: str) -> list:
    """
    预处理XML内容
    
    处理场景：
    1. 多流程文件（多个mxGraphModel拼接）
    2. 数据库导出带大量换行符
    3. 首尾空白字符
    """
    if not xml_content:
        return []
    
    # 1. 去除首尾空白
    xml_content = xml_content.strip()
    
    # 2. 统一换行符
    xml_content = xml_content.replace('\r\n', '\n').replace('\r', '\n')
    
    # 3. 去除标签间的多余换行
    xml_content = re.sub(r'>\n+<', '><', xml_content)
    
    # 4. 分割多流程（按<mxGraphModel>拆分）
    flow_list = re.findall(r'<mxGraphModel[^>]*>.*?</mxGraphModel>', 
                           xml_content, re.DOTALL)
    
    if not flow_list:
        # 单流程或格式异常，整体返回
        return [xml_content]
    
    return flow_list
```
